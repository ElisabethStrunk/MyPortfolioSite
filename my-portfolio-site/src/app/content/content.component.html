Connected Car
I did this project together with two friends for the "Laboratory for Microcontroller Technology" at the University Of Applied Sciences Rosenheim.
The aim of the project was to improve the position and motion estimation for a rolling ball on a two-axle traversable platform with an Extended Kalman Filter (EKF).
We first simulated the state estimation by EKF with real measurement data in MATLAB, then implemented it in C code, integrated it into an existing controller program and deployed it on a 32-bit microcontroller for real-time execution with the real system.
With our algorithm implemented there was an observable increase in the velocity and accuracy of the controller.
